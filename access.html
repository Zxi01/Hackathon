Accessibility Statement

Last updated: October 23, 2025

Commitment to Accessibility

We at Retro Arcade are committed to ensuring that our game and website are accessible to as many people as possible, including those with disabilities. We aim to adhere to applicable accessibility standards and guidelines to create an inclusive experience.

Measures Taken

  • The website uses semantic HTML and works with modern browsers, keyboard navigation, and assistive technologies (such as screen readers).
  • Game controls include keyboard support (arrow keys / WASD) and on-screen D-pad buttons for mobile or touchscreen users.
  • Text alternatives are provided for non-text content where appropriate, and color contrast has been optimized for readability.
  • The layout is responsive and adapts to both desktop and mobile screens.
  • Focus states on interactive elements are visible, and touch targets are large enough for easy interaction.

Known Limitations and Future Improvements

While we strive for full accessibility, the site may not yet meet all aspects of the WCAG 2.1 at level AA. Some current limitations include:

  • some visual elements rely on color; additional non-color cues are not yet provided for users with color vision deficiencies.
  • Animations and effects may cause discomfort for some users; an option to reduce motion will be added in future updates.
  • Keyboard focus order and labeling of all custom controls may require additional refinement for assistive technologies.
  • Gameplay changes (like tile merges or “Game Over”) may not always be fully announced by screen readers; we plan to improve ARIA live regions for better communication.

Feedback and Contact Information

If you encounter any accessibility barriers while using the game, please contact us so we can help you and improve our site.

  • Contact Us

When contacting us, please describe the issue, the page URL, and the browser or device you’re using so we can assist you more effectively.

Technical Specifications

Accessibility of this site depends on the following technologies:

  • HTML5 and CSS3
  • JavaScript (for game logic and controls)
  • Modern browsers such as Chrome, Firefox, Edge, and Safari

Assessment Approach

The accessibility of this site has been assessed using a combination of manual testing, mobile touch interface checks, and automated evaluation tools. We plan to conduct ongoing audits and improvements regularly.

Thank you for visiting and playing our game. We hope you have a fun and accessible experience!
